Script for task A is located in folder A, it is called BC_Logistic_regression_no_PCA.py, it executes logistic regression using breasmnist,npz dataset
Script for task B is located in folder B, it is called McC_Softmax_regression_PCA.py, it executes softmax regression using bloodmnist.nps dataset
Datasets folder is empty but prepared for the datasets breastmnist.np and bloodmnist.npz

Packages required to run this script are:
sklearn
Matplotlib
numpy
seaborn

To run the script run python main.py in terminal, select which script to run (A for script for breastmnist and B for script for bloodmnist) and press enter, to close the script press C.
